
# 🤖 Human Digital Twin Behaviour Prediction AI

> **AI-Powered Behaviour Analytics & Health Monitoring System**

An intelligent machine learning application that predicts a user's stress level based on daily lifestyle habits such as sleep, work hours, screen time, exercise, water intake, and daily steps. The application also provides a health score, personalized AI recommendations, interactive analytics, and downloadable health reports.


## 📌 Project Overview

Human Digital Twin Behaviour Prediction AI is a Machine Learning + Streamlit project designed to simulate a user's digital health profile.

The application analyzes daily behavioural patterns and predicts stress levels using a trained Machine Learning model.

It also generates:

- 🧠 Stress Prediction
- ❤️ Health Score
- ⚠️ Risk Level
- 💡 AI Recommendations
- 📊 Interactive Behaviour Analytics
- 📈 Weekly Health Trend
- 📄 Downloadable Health Report
